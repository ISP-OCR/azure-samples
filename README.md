# ISP OCR on Azure Samples
マイクロソフトAzure向けのOCRソリューション [ISP OCR on Azure](https://azuremarketplace.microsoft.com/ja-jp/marketplace/apps/issp-corporation.ispocr) 向けサンプル

# サポートOS
__Linux__  
- CentOS 6.9  
- Ubuntu 16.04  
__Windows__  
- Windows Server 2016  

# ビルド方法
__Linux__  
各サンプルのMakefileを使用してビルド  
__Windows__  
各サンプルのソリューション ファイルファイル(.sln)をVisual Studioから開いてビルド  

# Samples
|Module|Description|
|:--|:--|
|AccountingAnalyze|領収書OCR解析サンプル|
|LicenseAnalyze|自動車運転免許証OCR解析サンプル|
|MyNumberAnalyze|マイナンバーカードOCR解析サンプル|
|ReceiptAnalyze|レシートOCR解析サンプル|
